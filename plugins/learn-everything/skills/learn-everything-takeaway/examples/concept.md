# Example: concept takeaway (quality reference)

This is a constructed example for a concept explainer about "compound interest". Use as a quality target.

---

## Takeaway: 复利的力量——为什么爱因斯坦称它为"世界第八大奇迹"

**一句话总结：** 复利的反直觉之处在于，增长的主体不是本金而是"增长本身"，这使得时间而非金额成为财富积累的第一变量。

### 核心概念

复利（Compound Interest）：每个计息周期结束后，利息被加入本金，下一期以新本金计算利息。与单利的区别在于：单利只对本金计息，复利对"本金+已积累利息"计息。

公式：A = P(1 + r/n)^(nt)

### 核心原理（3条）

1. **时间是主变量，不是本金。** 在利率相同时，早投入10年比多投入一倍本金效果更显著。巴菲特95%的财富在60岁后积累，根源在此。
2. **增长加速度随时间增加。** 前期曲线平坦，后期陡峭——这与人类直觉（线性增长）相反，是复利最常被低估的原因。
3. **负向复利同样成立。** 信用卡债务、高利贷、坏习惯都以相同机制运作，只是方向相反。

### 常见误解

- ❌ "本金大才有意义" → 事实：时间 > 本金，年轻时小额投入优于中年大额投入
- ❌ "复利只适用于金融" → 事实：技能、人脉、知识积累都遵循复利逻辑
- ❌ "利率高才重要" → 事实：利率差异在短期不显著，在30年周期内可产生数倍差距

### 一个具体例子

甲：20岁起每年投入1万，投到30岁停止（共投10万），之后不再投入  
乙：30岁起每年投入1万，投到60岁（共投30万），年利率同为8%

60岁时：甲的账户约 **170万**，乙约 **120万**。甲少投20万，却多赚50万。

### 最值得记住的表达

> "复利是魔法，但它要求的代价是：你必须愿意等待无聊的事情变得惊人。"

### 这改变了我什么

复利框架让"现在做一件看似无用的事"变得理性——技能学习、每日写作、锻炼习惯，其价值不在当下，在于10年后的指数尾部。

## 什么使这个 takeaway 质量达标

- 核心概念有定义（不假设读者已懂）
- 三条原理每条都有"反直觉"维度，不是教科书列举
- 有具体数字例子（甲乙对比），不是抽象描述
- "最值得记住的表达"让概念有锚点，可复述
- 最后一段回答"这对我有什么用"，完成认知闭环
